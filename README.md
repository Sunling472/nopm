**Not Odin Package Manager**

Is Alpha version.

```bash
Command: get
Example: nopm get <repo_url> [flags]
Flags:
	-url:<string>, required    // git url
	-g                         // global install in share
	-op:<string>               // odin path

Command: new
Example: nopm new <project_name> [flags]
Flags:
	-name:<string>, required   // project name
	-path:<string>             // project path
```
